\# ML Project - Git \& GitHub Assignment



\## Project Files

\- train.py – Model training script

\- predict.py – Prediction script

\- utils.py – Helper functions

\- config.py – Configuration file

\- README.md – Project documentation



\## Collaborative Workflow Steps



1\. Checked project status  

&nbsp;  git status



2\. Staged local changes  

&nbsp;  git add utils.py config.py



3\. Committed changes  

&nbsp;  git commit -m "Update utils and add config file"



4\. Pulled teammate changes  

&nbsp;  git pull origin main



5\. Pushed updated project  

&nbsp;  git push origin main
